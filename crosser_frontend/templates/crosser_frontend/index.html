Hello!

Welcome to Crosser a program to explore plant breeding strategies. The user enters the donor plants and a crossing scheme, and Crosser calculates the proportion of the genome that comes from the recombinant parent, and the number of offspring that need to be tested to give a 95% chance of having 5 plants with the desired traits.

Crosser deals with averages, so the outputs are of the average situation. In real life, there is of course a range of possible outcomes around this.

Users of the program need to sign up for an account above and then contact the Trust Secretary or Deputy Trust Secretary with their user name for the account to be authorized before they can create and edit crosses. Guest users can view crosses and their outputs to the right.

Crosser is open source software built and maintained by the Kirkhouse Trust. You can find a copy of the crosser source code on Git Hub.
